### 1. Vertical Glare Divider Animation

A subtle animated background consisting of **vertical divider lines** with **blurred glowing dots**
(glare) moving upward infinitely.

#### Concept
- A centered container with a maximum width of `1200px`
- Two vertical lines dividing the layout into three visual sections
- Each line has a **glowing blurred dot** that moves from bottom to top
- Dark anthracite background for contrast
- Infinite loop animation

This background works especially well for:
- Hero sections
- Section separators
- Tech / futuristic UI designs
